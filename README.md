## Yolo Project useing for setup guide
### .NET
  * Clone
    ```bash
    git clone https://github.com/gunwoo8873/NET-DiscordBot.git
    ```
  * Restore
    ```bash
    dotnet restore
    ```
  * Build
    ```bash
    dotnet build
    ```
  * Run [Discord client run]
    ```bash
    dotnet run
    ```
